MyBox: Set of Easy Tools

This is a desktop program based on JavaFx, and its target to provide simple and easy functions.

Current version is 1.2. It includes the function: Convert each page of PDF file to an image with options of format, density, color, compression, quality, and color conversion. And user can pause/continue the conversion.

The following link is the complied Jar:

MyBox-1.2.jar

Under enrionment of JRE or JDK installed, run following command to launch this program:

                   java   -jar   MyBox-1.2.jar
