MyBox: Set of Easy Tools



This is GUI program based on JavaFx to provide simple and easy functions. It's free and open sources

Jar packages of each version have been uploaded at Release directory now. You can find them by clicking "releases" tab in main page of this project.

EXE package is avaliable for users who have not java env. It need not installation and users can run the EXE directly after unpack it.

MyBox-2.12-exe.zip

User also can download jar file if JRE or JDK installed, and run following command to launch this program:

                   java   -jar   MyBox-VERSION.jar

MyBox-2.12-jar.zip

Current version is 2.12. It includes the following features:

  1. Switch interface in Chinese and English at any time.
  2. Convert each page of PDF file to an image with options of format, density, color, compression, quality, and color conversion.
  3. Extract images in PDF and save as original format.
  4. Extract texts from PDF files, with customized page separator.
  5. Combine images as PDF file
  6. Compress images in PDF
  7. Combine PDF files
  8. Split PDF
  9. Support Chinese written in PDF file, and system font file is locating automatically while user can input ttf file path.
  10. Convert image files to other formats, with options of color, size, compression, quality, etc.
  11. Above functions can be run in batching way. User can pause/continue the process.
  12. Supported image formats: png, jpg, bmp, tif, gif, wbmp, pnm, pcx.
  13. Manufacture image: Adjust pixels, saturation, lightness, hue, and RGB channels with parameters. Provide filters like gray, invert, binary, and RGB channels. Implement croping, color replacement, opacity adjustment, cutting edges, adding edges, adding watermark, and transforms like rotating, mirrorings, and shearing. Support images browsering and reference image.
  14. Implement manufacture of big image and make sure performance is acceptable.
  15. Scope, including area scope like rectangle and circle and color matching, is supported for image manufacture.
  16. View information and meta data of image files.
  17. Image Spliting which supports equipartition and custom.
  18. Image combining which supports array options, background color, interval, edges, and size options.
  19. View multiple image files in same screen. Rotate and zoom images separately or at same time. Support multiple images browsering.
  20. Color Palette. Pixels Calculator.
  21. Rename files under directories, with options about files' name and sorting. All of or part of renamed files can be recovered as originl names.
  22. Directory synchronization, with options like copying subdirectoies, new files, modified files after some time, original file's attributes, or deleting files and directories which are not in source path, etc.
  23. Files rearragement that categories files under new directories according to their modify time or create time. This function can be used to handle photoes, games screenshots, or system logs which need archived based on time.
  24. Alarm clocks, with options of time and ring. Support rings of "Meow", wav, mp3. Can run in background.
  25. Html Editor and Text Editor. Web browser supports synchronized contents with web editor.
  26. Web page can be snapped in one image or in PDF file with settings of delay and PDF page size.
