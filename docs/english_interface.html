MyBox: Set of Easy Tools

This is a desktop program based on JavaFx, and its target to provide simple and easy functions.

Current version is 1.7. It includes the functions:

1) Convert each page of PDF file to an image with options of format, density, color, compression, quality, and color conversion. And user can pause/continue the conversion.

2) Extract images in PDF and save as original format.

3) Extract images and convert to images from PDF files in batching way.

4) Convert image files to other formats, with options of color, size, compression, quality, etc. Supported formats: png, jpg, bmp, tif, gif, wbmp, pnm, pcx.

5) View information and meta data of image files.

6) View multiple image files in same screen. Rotate and zoom images separately or at same time.

7) Pixels Calculator.

Jar packages of each version have been uploaded at Release directory now. You can find them by clicking "releases" tab in main page of this project. The latest version is:

MyBox-1.7.jar

Under enrionment of JRE or JDK installed, run following command to launch this program:

                   java   -jar   MyBox-VERSION.jar
