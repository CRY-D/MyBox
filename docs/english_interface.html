MyBox: Set of Easy Tools

This is GUI program based on JavaFx to provide simple and easy functions. It's free and open sources

Jar packages of each version have been uploaded at Release directory now. You can find them by clicking "releases" tab in main page of this project.

EXE package is avaliable for users who have not java env. It need not installation and users can run the EXE directly after unpack it.

MyBox-4.1-exe.zip

User also can download jar file if JRE or JDK installed:

MyBox-4.1-jar.zip

Run following command to launch this program:

                   java   -jar   MyBox-4.1.jar

Current Version

Current version is 4.1. It includes the following features: 

1. PDF Tools:
	A. Convert each page of PDF file as a image, with options like density, color model, format, compression, quality, color conversion, etc. 
	B. Combine multiple images into PDF file, with options like compression, page size, margins, page header, author, etc. 
	   Support Chinese and program can locate system font files. User can input the ttf font file path too.
	C. Compress images in PDF file, with options like JPEG quality or black-white threshold.
	D. Merge multiple PDF files.
	E. Split PDF file into multiple PDF files, by pages number, files number, or start-end list.
	F. Extract images from PDF file. Can define the page range.
	G. Extract texts from PDF file. Can customize the deviding lines.
	H. Handle PDF files in batch way.
	I. Set the Maximum main memory of PDF handling.
2. Image Tools:
	A. Increase/Decrease: Pixels size, saturaion, brightness,  hue, Red/Green/Blue/Yellow/Cyan/Magenta channel.
	B. Filters: Gray, black-white, Sepia, invert, Red/Green/Blue invert, Red/Green/Blue/Yellow/Cyan/Magenta channel
	C. Effects: Blur, sharpen, emboss, edges detect, posterize(reduce colors), thresholding.
	D. Crop, replace color,  watermark, cover, arc, shadow, shear, vertical/horizantal mirror, rotate, margins, and convolution.
	E. Define “scope”: Area(Rectrangle or circle) and color matching. Used for handling image in part.
	F. “Undo” and “Redo” of previous operation. Can recover original image at any time. 
	   Can record some updating histories of image manufacture, to return to former updating. User can set the number of updating histories.
	G. Make sure correctness and performance of manufacture of big image.
	H. Browse images under same directory.
	I. Select whether show reference image. Can select other pictures as the reference image.
	J. View the image meta-data and image attributes.
	K. Split the image. Support equal splitting and customized splitting.
	L. Combine images. Support options like array ordering, background color, interval, margins, and size.
	M. Blend images. Support options like defining intersect area, whether keep intersect area only, and multiple famous blending modes.
	N. View multiple images in same screen. Support rotating and zooming at same time. Support browsing.
	O. Convert image into other format, with options like color model, size, compression, quality, etc.
	P. Handle images in batch way.
	Q. Color palette
	R. Pixels calculator
	S. Convolution Kernels Manager
3. Desktop Tools:
	A. Rename Files/Directories, with options of files' name and ordering.  Renamed files can be recovered as original names in all or in part.
	B. Sychronize directories, with options like whether copy sub-directories or new files, whether only copy modified files after specific date time, 
	   whether keep attributes of original files, or whther delete non-existed files/directories under original directory.
	C. Arrange files and reorganize them under new directories by modifed time. 
	   This tool can be used to handle lots of files which need be archived according to time, like photoes, screenshots of games, or system logs.
	D. Text editor.
	E. Alarm clocks, including options of time and music. Support rings of “Meow”, wav files, and mp3. Can run in background.
4. Network Tools:
	A. Html Editor
		a. Edit local web pages or online pages in rich text.
		b. Code Editor can be sychronized with Html Editor.
		c. Web browser can be sychronized with contents of Editors, or load the online page. 
		   Support history browsing, font zooming 、and snapshoting of the whole page as a image or a PDF file.
	B. Weibo Snaping Tool
		a. Save Weibo pages of any months of any Weibo accounts automatically.
		b. Set the months range.
		c. Make sure whole page contents loaded. Can expand the comments and pictures in the pages.
		d. Save the pages as local html files which can not be loaded normally due to dynamic loading of WeiBo contents. 
		   They can be used to extract texts in the pages.
		e. Save the pages' snapshots as PDF files, with options like page size, margins, author, images format, etc.
		f. Save all original size pictures in the pages.
		g. Display progress information in time.
		h. Stop the progress at any time. The interrupted month will be record and input as start month for next execution.
		i. Set the retry times of failure.
5. Supported image format: png,jpg,bmp,tif,gif,wbmp,pnm,pcx.
6. Settings: Languages(Chinese and English), whether show comments, maximum main memory used for PDF, how to handle alpha channel of image, 
   whether close alarm clocks when exit program,  clearing settings,  and more parameters.
7. Multiple interface styles.
