MyBox：简易工具集

这是利用JavaFx开发的图形化界面程序，目标是提供简单易用的功能。免费开源。

项目主页：https://github.com/Mararsh/MyBox

每个版本的源代码和编译好的包已发布在Release目录下：https://github.com/Mararsh/MyBox/releases

欢迎在线提交软件需求和问题报告：https://github.com/Mararsh/MyBox/issues

可以下载exe包，无需java环境、无需安装、解包可用：MyBox-2.5-exe.zip

在已安装JRE或者JDK的环境下，可以下载jar包：MyBox-2.5-jar.zip 执行以下命令来启动程序：

     java   -jar   MyBox-版本号.jar
